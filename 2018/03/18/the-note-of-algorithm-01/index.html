算法笔记》学习笔记（一） c/c++快速入门

Posted on 2018-03-18 | In 计算机 | | Views:

基本数据类型

变量类型

  • 整型
    • int
      • $10^9$以内可定义为int
      • 实际范围 - $-2^{31}-+({2^{31}-1})$
    • long long
      • 取值超过$2^{63}-1$时可用
  • 浮点型
    • float
      • 6-7位有效精度
    • double
      • 15-16位有效精度
  • 布尔型
  • 字符型
    • 小写字母比大写字母ASCII码值大32

常量

  • 宏定义

  • const常量

顺序结构

输入与输出

  • scanf(“格式控制”, 变量地址);
    • double用%lf读入
    • 以空白符判断结束
      • 空格
      • 换行
  • printf(“格式控制”, 变量名称);

  • 实用输出格式

    • %md
      • 使不足m位的整数以m位输出，不足位数用空格填充

    • %0d

      • 与上同，但以0填充

    • %.mf

      • 控制浮点数位数
      • 当题目要求是保留到小数点后XX位时使用

常用math函数

  • fabs(double x)
  • floor(double x)和ceil(double x)
  • pow(double r, double p)
    • 返回$r^p$
  • sqrt(double x)
  • log(double x)
  • sin(double x)
  • asin(double x)
  • round(double x)

数组

  • 若数组空间较大，定义在主函数外

    • eg：$10^6$大小

  • 使用memset对数组中每一个元素赋相同的值

    • 只建议初学者用其赋0或-1
    • 在string.h头文件中
  • 字符数组
    • 如果用%s输出，需加上\0
    • sscanf(str, “格式控制”, 变量地址)
      • 从字符数组到变量
    • sprintf(str, “格式控制”, 变量名称)
      • 从变量到字符数组

指针

  • 数组名称是指向数组首地址的指针
  • 存储变量地址

引用

  • 给变量起别名
  • 只能引用变量
  • 可引用指针

浮点数的比较

  • 由于浮点数不是精确存储，所以引入一个$\epsilon$来确定误差
  • $\epsilon$通常为$10^{-8}$
    • 用科学记数法表示为1e-8

时间复杂度

  • OJ系统一般一秒可承受的运算次数为$10^7$-$10^8$
    • $O(n^{2})$级的算法规模为1000时可承受，n为100000时是不可接受的

黑盒测试

  • 单点测试
    • 根据每组输入数据的结果给出对应分数
    • pat采用
  • 多点测试
    • 要求一次能运行所有数据
    • 大部分在线OJ采用
    • 例
      • while···EOF型
      • while···break型
      • while···T型
